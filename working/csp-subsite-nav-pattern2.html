Pattern 2 — implementation notes

CTA slot

Each subsite defines its own primary action in one place — the .csp-sub-cta anchor and its matching .csp-sub-mobile-cta. Button color inherits from the accent class automatically — nothing extra to configure per subsite.

Mobile behavior

Below 768px the link cluster hides and a toggle appears. The collapse panel uses a max-height CSS transition (no JS animation needed) and exposes full-width tap targets ≥ 44px. The CTA sits at the bottom of the panel — prominent and thumb-friendly.

Adding a new subsite

Copy any nav block, swap the accent class (accent-teal / accent-gold / accent-navy), update subsite name, icon, links, and CTA text. A fourth accent (e.g. for a Tutoring subsite) is just 5 new CSS rules targeting .accent-purple.

LibGuides placement

The <nav> block goes in each guide group's Custom Header. The <style> and <script> live in the system-wide Custom CSS / JS fields — one shared copy, no duplication across guide groups.
